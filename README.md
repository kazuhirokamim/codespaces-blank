# Análise Estatística do Dataset Iris

Este projeto foi desenvolvido como atividade da disciplina do curso de Análise e Desenvolvimento de Sistemas (ADS) da matéria Ciência de Dados. O objetivo é realizar uma análise estatística simples utilizando o famoso dataset Iris.

## Sobre o Dataset

O dataset Iris é um dos conjuntos de dados mais conhecidos em Ciência de Dados e aprendizado de máquina. Ele contém medições de flores da espécie Iris, divididas em três classes:

* Setosa
* Versicolor
* Virginica

As variáveis analisadas são:

* comprimento da sépala (sepal length)
* largura da sépala (sepal width)
* comprimento da pétala (petal length)
* largura da pétala (petal width)

## Objetivo da Atividade

Realizar cálculos estatísticos utilizando Python para analisar as variáveis do dataset e criação de gráficos como Histograma, Boxplot, Scatter Plot, ANOVA e Tukey.

Entre as análises realizadas estão:

* média
* mediana
* variância
* desvio padrão
* análise de atributos específicos do dataset

## Tecnologias Utilizadas

* Python
* Pandas
* Matplotlib
* Pingouin
* Dataset Iris (scikit-learn)

## Como executar

1. Instale as bibliotecas necessárias com o comando:
pip install pandas matplotlib pingouin scikit-learn

2. Abra o notebook da atividade em um ambiente que suporte Python, como:
* Jupyter Notebook
* VS Code
* Google Colab

3. Execute as células do notebook na ordem em que aparecem para carregar os dados, gerar os gráficos e realizar as análises estatísticas.
